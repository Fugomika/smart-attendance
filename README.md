# Smart Attendance

📌 **Kelompok 6 – Proposal Ide TUBES Aplikasi Berbasis Platform**

Aplikasi absensi berbasis Android dengan fitur:
- **GPS validation** untuk memastikan lokasi absensi.
- **Face recognition** sebagai autentikasi pengguna.
- **Real‑time attendance data** yang langsung tersimpan.
- **HR dashboard** untuk monitoring & analisis data kehadiran.

---

## 🎯 Problem Statement
- Absensi manual rawan kecurangan (titip absen, manipulasi waktu).
- Data tidak real‑time, sulit dipantau.
- HR kesulitan rekap & analisis.

## 💡 Solution
- Mobile app dengan integrasi GPS + face recognition.
- Dashboard HR untuk visualisasi & analisis.
